# University Management System

Spring Boot application for managing students, teachers, courses, and users in a university. Includes both a REST API (JWT-secured) and a Thymeleaf-based web UI.

## Tech Stack

- **Java 17**, **Spring Boot 3.5**
- **Spring Security** + **JWT** authentication
- **Spring Data JPA** + **MyBatis**
- **PostgreSQL**
- **Thymeleaf** (server-side UI)
- **SpringDoc OpenAPI** (Swagger UI)
- **Lombok**

## Prerequisites

- Java 17+
- Maven 3.8+
- PostgreSQL 14+

## Database Setup

Create a PostgreSQL database:

```sql
CREATE USER lab2 WITH PASSWORD 'lab2';
CREATE DATABASE lab2_db OWNER lab2;
```

The schema is auto-generated by Hibernate (`ddl-auto=update`).

## Configuration

Key properties in `src/main/resources/application.properties`:

| Property | Default | Description |
|---|---|---|
| `spring.datasource.url` | `jdbc:postgresql://localhost:5432/lab2_db` | Database connection URL |
| `spring.datasource.username` | `lab2` | Database username |
| `spring.datasource.password` | `lab2` | Database password |
| `jwt.secret` | `my-super-secret-key-...` | JWT signing key |
| `jwt.expiration-ms` | `86400000` (24h) | JWT token lifetime |
| `spring.servlet.multipart.max-file-size` | `5MB` | Max upload file size |

## Running the Application

```bash
./mvnw spring-boot:run
```

The app starts on `http://localhost:8080`.

## API Documentation (Swagger)

Once running, open:

- **Swagger UI**: [http://localhost:8080/swagger-ui.html](http://localhost:8080/swagger-ui.html)
- **OpenAPI JSON**: [http://localhost:8080/v3/api-docs](http://localhost:8080/v3/api-docs)

To test authenticated endpoints in Swagger UI:
1. Call `POST /auth/login` or `POST /auth/register`
2. Copy the `token` from the response
3. Click **Authorize** and enter: `<your-token>`

## API Endpoints

### Authentication (`/auth`) -- public

| Method | Path | Description |
|---|---|---|
| POST | `/auth/login` | Login and get JWT token |
| POST | `/auth/register` | Register a new account |
| PATCH | `/auth/change-password` | Change password (authenticated) |

### Users (`/api/users`) -- ADMIN only

| Method | Path | Description |
|---|---|---|
| GET | `/api/users` | Search/paginate users |
| POST | `/api/users` | Create user |
| PUT | `/api/users/{id}` | Update user |
| DELETE | `/api/users/{id}` | Delete user |

### Students (`/api/students`) -- ADMIN only

| Method | Path | Description |
|---|---|---|
| GET | `/api/students` | Search/paginate students |
| POST | `/api/students` | Create student |
| PUT | `/api/students/{id}` | Update student |
| DELETE | `/api/students/{id}` | Delete student |

### Teachers (`/api/teachers`) -- ADMIN only

| Method | Path | Description |
|---|---|---|
| GET | `/api/teachers` | Search/paginate teachers |
| POST | `/api/teachers` | Create teacher |
| PUT | `/api/teachers/{id}` | Update teacher |
| DELETE | `/api/teachers/{id}` | Delete teacher |

### Courses (`/api/courses`) -- ADMIN only

| Method | Path | Description |
|---|---|---|
| GET | `/api/courses` | Search/paginate courses |
| POST | `/api/courses` | Create course |
| PUT | `/api/courses/{id}` | Update course |
| DELETE | `/api/courses/{id}` | Delete course |

### Profiles (`/api/profiles`) -- authenticated

| Method | Path | Description |
|---|---|---|
| POST | `/api/profiles/{userId}/avatar` | Upload avatar |
| POST | `/api/profiles/{userId}/documents` | Upload documents |
| GET | `/api/profiles/files/{fileId}` | Download file |
| DELETE | `/api/profiles/files/{fileId}` | Delete file |

## User Roles

| Role | Access |
|---|---|
| `GUEST` | Default role after registration |
| `STUDENT` | Authenticated access |
| `TEACHER` | Authenticated access |
| `ADMIN` | Full CRUD access to all `/api/*` endpoints |

## Project Structure

```
src/main/java/kz/iitu/hello/
├── config/             # DataInitializer, OpenAPI config
├── domain/
│   ├── entity/         # JPA entities (User, Student, Teacher, Course, UserFile)
│   ├── enums/          # UserRole, Department
│   ├── mapper/         # MyBatis mappers
│   ├── repository/     # Spring Data JPA repositories
│   └── specification/  # JPA Specifications for search
├── exception/          # Custom exceptions + global handler
├── security/           # JWT filter, SecurityConfig, UserDetailsService
├── service/            # Business logic layer
└── web/
    ├── controller/
    │   ├── api/        # REST controllers
    │   ├── auth/       # Authentication controller
    │   └── mvc/        # Thymeleaf controllers
    ├── converter/      # Entity <-> DTO converters
    ├── dto/            # Request/response DTOs
    └── validations/    # Custom validators
```
